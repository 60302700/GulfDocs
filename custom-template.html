Line Items Table Repeating Loop
Wrap your row elements inside {{#items}} block. Our target engine repeats this segment for each item cards.
{{#items}} <tr> <td>{{item_index}}</td> <td>{{item_description}}</td> <td>{{item_sku}}</td> <td>{{item_qty}}</td> <td>{{item_price}}</td> <td>{{item_total}}</td> </tr> {{/items}} Dynamic Repeating row block syntax
{{items_table}} No time? Just paste this for a quick pre-built table.
